## 🐛 Troubleshooting

### Animations not working?
- Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Check browser console for errors
- Try Chrome/Firefox/Safari latest

### Hover effects not visible?
- Ensure hovering over card, not just text
- Try different browser
- Check if on touch device

### Particles not visible?
- They're subtle! Look carefully
- Small white dots floating up
- More visible in certain lighting

### Layout looks wrong?
- Clear browser cache
- Hard refresh page
- Check browser width
